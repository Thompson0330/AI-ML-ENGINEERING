# Number Guessing Game 

## Description
This is a simple python game where the computer selects a random  number and the user tries to guess it

## Features
- Random number generation
- User input handling

## Technologies used
- Python

## How to run 
1. Install python
2. Open Terminal
3. Run:
    python guess_game.py

## What i learned
- Loops (while)
- If/else statements
- User input in python
